ELF Application Binary Interface

s390x Supplement

Official Specification Document

This document specifies the ELF Application Binary Interface (ABI) for the s390x architecture (IBM z/Architecture). It defines the low-level interface between application programs, libraries, and the operating system.

About This Specification

The s390x ELF ABI specification is maintained by IBM and the open source community. It provides the foundation for binary compatibility across different compilers, libraries, and operating systems on the s390x platform.

The specification is built from LaTeX source files available in the GitHub repository. Contributions and feedback are welcome through GitHub issues and pull requests.
